# Live game overrides

This folder is the active Git-managed layer for the deployed Thousandfold Realms game.

GitHub Pages unpacks the latest web package, then automatically injects every `.css` and `.js` file in this folder into the playable `index.html` and `404.html`.

Use this folder for live gameplay, interface, bug-fix, and balance changes so commits deploy automatically without rebuilding or manually uploading the large music/assets package each time.

Files load alphabetically. Prefix dependent patches with ordered numbers when load order matters.
