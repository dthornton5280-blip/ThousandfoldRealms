# Thousandfold Realms generated runtime art

The files in this directory are project-specific runtime derivatives created for Thousandfold Realms.

## `thousandfold-haven-v161.png`

- Original generated pixel artwork commissioned for this project.
- Built from the approved Haven exterior and Black Lantern Tavern art-direction concepts.
- Cleaned, cropped, transparency-masked, and packed into a purpose-built runtime atlas.
- Contains only assets intended for the game runtime; presentation labels, catalog borders, and unused concept-sheet material were excluded.
- Uses a 16-pixel source-art discipline while rendering on the game's 32-pixel gameplay grid.

The atlas is not a redistributed copy of the Pixel Crawler pack. Pixel Crawler remains separately credited under `source/assets/third-party/pixel-crawler/NOTICE.txt` and is used only according to the project rules in `docs/PIXEL_CRAWLER_ART_DIRECTION.md`.
