# One-time GitHub Pages deployment

1. Download `Thousandfold_Realms_Web_v1.4.4-dev.zip` from the ChatGPT development chat.
2. In this repository, choose **Add file → Upload files**.
3. Upload the ZIP to the repository root without renaming it.
4. Commit directly to `main`.
5. Open **Settings → Pages** and select **GitHub Actions** as the source.
6. Open the **Actions** tab and wait for **Deploy Thousandfold Realms** to finish.

The workflow unpacks the full game, including the user-supplied music, and deploys it as a GitHub Pages site. The expected URL is:

`https://dthornton5280-blip.github.io/ThousandfoldRealms/`

The repository must be public for Pages on a free personal GitHub account. Confirm the music track's redistribution and attribution rights before treating the site as a public release.
